# Project Bootstrap Kit

A reusable scaffold for bootstrapping **Claude Opus 4.7** onto new projects in approximately 5 minutes. Clone this kit as `_bootstrap/` into any new project root and follow the steps in `CLONE_AND_USE.md` to get a fully onboarded AI co-developer up and running immediately.

**Default stack:** React 19 + Vite + Express 5 + Prisma 6 + Postgres 16, hosted on Fly.io (Frankfurt) + Netlify + Neon.

**Methodology:** [Ralph](https://github.com/snarktank/ralph) + [Caveman](https://github.com/JuliusBrussee/caveman) + full memory/run-log discipline.

## How to use

1. Clone this repo into your project (see `CLONE_AND_USE.md` for the full workflow).
2. Open `_bootstrap/00_SYSTEM_PROMPT_BOOTSTRAP.md` and fill in the 4 mandatory placeholders.
3. Paste the prompt block into a fresh Claude Opus 4.7 session.

> **Note:** The 7 bootstrap content files (`00_SYSTEM_PROMPT_BOOTSTRAP.md` through `06_PLATZHALTER_AUSFUELLEN.md`) are provided by the owner as a separate upload and placed inside `_bootstrap/`. The file `_bootstrap/README.md` contains a detailed list of expected files and their source location.
